<div align="center" id="starting">
  <h1>Starting</h1>

  *Clone the project and access the folder*

  ```bash
  $ git clone https://github.com/alisonsantosofc/barbero-api
  $ cd barbero-api
  ```

  *Install the dependencies*

  ```bash
  $ yarn
  ```

  *Start the project in development environment*

  ```bash
  $ yarn start
  ```
</div>
